G-CASPER

Call Sheet & Production Report System  |  Standalone Edition

⚙️

1. Set Your Globals

Start on the Globals tab. Enter your show name, key crew, SAG info, walkie channels, and hot-cost rates.

📅

2. Add Shoot Days

Use the Days tab to list every day of your schedule with date, nickname, and location.

🎬

3. Build Your Cast & Breakdown

Add cast members on Cast and enter scene details on Breakdown.

📋

4. Generate Call Sheets

The CS Front/Back tabs (yellow group) auto-populate from your globals, cast, and breakdown data.

📊

5. Production Reports

Exhibit G, PR Front & Back (green group) track SAG times, stats, crew hours, and meals.

💾

6. Save & Load

Use Save to download a .casper project file. Use Load to reopen it any time.

Collapsible Tab Groups: The yellow Call Sheet group (CS Front + CS Back) and the green Prod. Report group (Exhibit G + PR Front + PR Back) can each be collapsed into a single combined tab using the ⇄ button. Click again to expand.
